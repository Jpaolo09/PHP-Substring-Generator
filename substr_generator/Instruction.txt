Paano gamitin?

SETUP:
1. lagay niyo tong buong folder na to (e.g. "substr_generator") sa "htdocs" folder ng xampp
2. Open xampp then start apache
3. Open browser
4. Go to http://localhost/substr_generator/substr_generator.php

GENERATE CODE:
1. Open niyo tong file na to
2. Lagay niyo ung word na gusto niyong gawan sa loob ng word variable (line 3)
3. Save / Ctrl + S
4. refresh browser
5. tadah! copy paste niyo nlng ung code na ma gegenerate
